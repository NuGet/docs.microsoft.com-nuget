# NuGet Warning NU1602

> 'PackageA' 4.0.0 does not provide an inclusive lower bound for dependency 'PackageB' (> 3.5.0). 3.6.0 was resolved instead.

### Issue

A package dependency is missing a lower bound.
This doesn't allow restore to find a minimum version to use.

### Solution

This is usually a package authoring error.
Contact the package author to resolve the issue.
